#Backend:

	- Spring Boot (Java 11)

#Frontend:

### Indítás
1. mvn clean install in sporttracking-backend directory
2. mvn spring-boot:run in sporttracking-backend directory
3. yarn install in sporttracking-app directry
4. yarn start in sporttracking-app directory
